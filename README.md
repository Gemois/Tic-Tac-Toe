# Tic-Tac-Toe
This project is a simple implementation of the game Tic-Tac-Toe using python.


## How To Run:

```
git clone https://github.com/Gemois/Tic-Tac-Toe.git
cd Tic-Tac-Toe
python TicTacToe.py
```
